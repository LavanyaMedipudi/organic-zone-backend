## Description
Organic Zone is a Java-based backend application for an e-commerce platform.
The project focuses on backend development using Spring Boot, RESTful APIs, and MySQL database integration.

## Features
- REST API for product management
- Health check API to verify backend status
- Layered architecture (Controller, Service, Repository)
- Database interaction using Spring Data JPA

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Project Structure
- Controller layer for handling REST API requests
- Service layer for business logic
- Repository layer for database access
- Model layer for entity representation
